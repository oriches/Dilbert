Dilbert
=======
A .Net class library to access the Daily Dilbert cartoon.

Supported versions:

    .NET framework 4.0 and higher


This class library makes use of the 'async & await' keywords available in the standard install of .Net 4.5 and by the inclusion of the <a http="https://www.nuget.org/packages/Microsoft.Bcl.Async">Microsoft.Bcl.Async</a> NuGet package for .Net 4.0, this means .Net 4.0 applications wanting to use this will have to have the Microsoft Hotfix <a href="http://support.microsoft.com/kb/2468871">KB2468871</a> installed.
